# WINDOWS ENVIRONMENT SETUP NOTES
################################################################################
# 
# 1. Poppler (Required for pdf2image to convert PDF pages to images)
#    - Linux: Can be installed via `sudo apt-get install poppler-utils`.
#    - Windows: 
#      a. Download the latest Windows binaries of Poppler (e.g., from: 
#         https://github.com/oschwartz10612/poppler-windows/releases/ )
#      b. Extract the downloaded ZIP folder (e.g., to C:\Program Files\poppler).
#      c. Add the `bin/` directory (e.g., `C:\Program Files\poppler\Library\bin`)
#         to your Windows System Environment variables (PATH).
#      d. Alternatively, pass the path directly in your python ingestion code:
#         `convert_from_path(..., poppler_path=r"C:\Program Files\poppler\Library\bin")`
#
# 2. Libmagic (Required for unstructured library file type detection)
#    - Linux: Usually works natively or via `sudo apt-get install libmagic-dev`.
#    - Windows: `libmagic` does not come natively. You need to install the Windows port:
#      `pip install python-magic-bin`
#
# 3. Psycopg2 (Database connection)
#    - Windows & Linux: We use `psycopg2-binary` instead of `psycopg2`. 
#      This avoids compiling from source and does not require PostgreSQL C-headers 
#      or a C++ compiler to be installed locally.
#
# 4. Tesseract OCR (Optional, required if unstructured needs OCR on scanned PDFs)
#    - Windows: Download and install the Tesseract installer from UB Mannheim.
#    - Add `C:\Program Files\Tesseract-OCR` to the system environment variables PATH.
#
# 5. C++ Build Tools (Required if any dependency tries to compile wheel extensions)
#    - Windows: If you encounter compiler errors, install Visual Studio Build Tools 
#      with the "Desktop development with C++" workload selected.
#
